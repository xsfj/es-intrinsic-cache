# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `es-intrinsic-cache`, please **open a GitHub issue** in this repository. When reporting, include:

- A clear description of the issue
- Steps to reproduce the problem
- Any potential impact or affected functionality

We will review issues promptly and work to address them as quickly as possible.

## Supported Versions

Only the latest release (`1.0.x`) of `es-intrinsic-cache` is actively supported for security fixes.  

## Security Updates

Confirmed security issues will be tracked via GitHub issues and patched in the next release. Users are encouraged to update to the latest version promptly.
